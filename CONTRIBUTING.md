# Contributing to Vaartha

We welcome contributions! However, to protect the project's future, we require all contributors to agree to our CLA.

## 1. Legal Requirements
Before we can merge your Pull Request, you must agree to the **[Contributor License Agreement](CLA.md)**.
*   By submitting a PR, you automatically agree to the terms in `CLA.md`.

## 2. Coding Standards
*   **Android**: Follow Kotlin Style Guide.
*   **Desktop**: Follow PEP8 for Python.
*   **Commits**: Use conventional commits (e.g., `feat: login screen`, `fix: crash on startup`).

## 3. Workflow
1.  Fork the repo.
2.  Create a branch (`feat/my-feature`).
3.  Commit and Push.
4.  Submit a Pull Request.
